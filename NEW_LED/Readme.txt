	Led event를 두 종류로 구분한다
		1) One-shot event
			- 이벤트 발생 시점에 사용자에게 알려주기 위한 용도
			- 전화 수신, Notification 수신, 알람 발생, 모션인식 등 발생 시점에 알려주어야 의미가 있는 이벤트에 적합
			- 발생 시점이 아닌 몇초 뒤 보여주는 것은 의미가 없음
			- 가능한 동작 : 복합적 패턴 플레이(깜빡인다든지 여러가지 색을 번갈아가며 보여준다던지 모두 가능)
			
		2) Periodic event(On-going event)
			- 지속성이 있는 이벤트
			- 단말의 상태를 나타내는데 적함함
			- 배터리 상태, 미확인 메세지 유무 상태 등
			- 가능한 동작 : Solid on, 주기적으로 깜빡임(컬러를 바꿔가면서 보여준다던지, 깜빡이는 주기가 가변적인 동작은 불가능)
